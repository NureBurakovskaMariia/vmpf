# vmpf

ПЗ 1(реалізовано на максимальний бал, по завданню з кожного рівня:

Запитайте користувача про його ім'я та виведіть привітання з використанням введеної назви.
Створіть список чисел від 1 до 20. Виведіть на екран кожне число та його квадрат.
Створіть функцію, яка приймає список чисел та повертає новий список, який містить лише парні числа
Реалізуйте програму для роботи з API. Використовуйте бібліотеку requests, щоб отримати дані з публічного API та вивести їх на екран.
